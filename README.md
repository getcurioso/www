# Curio TCG Website

Landing page for Curio TCG - Your ultimate Pokemon Trading Card Game collection manager.

## Overview

Curio TCG is a mobile app that helps collectors organize, trade, and showcase their trading card collections with powerful tools designed for serious collectors.

## Features

- **Manage Collections** - Track cards with condition, variant, and quantity details
- **Create Binders** - Organize cards into custom digital binders
- **Marketplace** - Buy and sell cards with other collectors
- **Wishlist** - Track cards you want to acquire
- **Graded Cards** - Catalog PSA, BGS, CGC, and SGC graded cards
- **Social Features** - Like, share, and connect with the community
- **Browse Sets & Series** - Explore complete card sets with detailed info

## Development

To run locally, simply open `index.html` in a browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```

Then visit `http://localhost:8000`
